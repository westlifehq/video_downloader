# 项目上下文 (Project Context)

## 项目目标
构建一个全平台（目前支持抖音、小红书）的高画质、无水印视频下载工具，支持本地 PC（Mac/Windows）运行及 NAS（Docker）部署。重点解决用户在移动端发现素材后，能够快速触发服务器或本地电脑进行高清原图/原片归档的需求。

## 核心功能
- **抖音收藏夹同步 (v2)**：集成 Playwright 浏览器自动化，支持扫码登录后自动同步最近 50 条收藏视频，支持手动下载、批量下载、已下载状态标识及本地文件管理（打开文件夹/物理删除）。
- **跨平台解析**：支持 **抖音 (Douyin)** 和 **小红书 (Xiaohongshu)** 视频内容解析，自动识别平台并切换引擎。
- **无水印原画质**：直取底层接口，下载官方无压缩、无水印的 1080P/720P 原视频。
- **自动短链追踪**：完美支持分享短链接，自动识别并跟随 302 重定向。
- **极速批量处理**：支持在输入框同时粘贴多条链接，自动剥离文字，多线程并行解析与下载。
- **高级定制 UI v2**：采用全新的 **Premium Minimalist (紫/靛蓝)** 暗色毛玻璃设计，视觉更高级。
- **跨平台一键部署**：无论是 Mac、Windows 还是 Linux，仅需 Node 环境即可轻松跑满带宽。

## 部署模式
1. **本地运行 (Mac/Windows)**：直接通过 `npm run dev` 启动，或者双击 `.command` / `.exe` 文件。支持调用本地文件管理器定位下载的文件。
2. **服务器化/NAS 部署 (nas-deployment)**：独立的 `nas-deployment` 文件夹包含了支持 Docker 环境的代码。可以直接将其构建为 Docker 镜像，部署在飞牛 OS (fnOS) 等 x86 NAS 上。该版本包含了针对远程环境、移动端浏览器的特殊优化。
3. **外网访问优化 (DDNSTO)**：通过飞牛原生 Docker 部署了 DDNSTO 穿透服务，实现外网通过 HTTPS 域名直接控制 NAS 的下载任务。

## 技术栈
- **后端**：`Node.js`, `Express`, `Axios (0.27.2)`, `Playwright`
- **前端**：`Vanilla JS`, `Vanilla CSS`, `HTML5`
- **打包**：`pkg`
- **穿透**：`DDNSTO` (Docker)
