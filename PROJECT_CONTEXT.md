# 项目上下文 (Project Context)

## 项目简介
Video Downloader 是一款基于 Node.js 构建的全平台（目前支持抖音、小红书）无水印视频/图文下载器。
分为前后端架构，后端负责接口解析、反爬策略应对与大文件下载，前端负责用户交互。

## 核心功能
- **平台支持**：支持解析抖音、小红书的视频、图集链接，自动追踪短链接重定向。
- **解析与下载**：调用底层无水印接口拉取原画质内容，并通过本地服务器直接下载存储至指定目录。
- **本地历史管理**：支持查看下载历史，包括文件大小、缩略图，提供在操作系统（Finder/资源管理器）中直接定位文件，以及从物理磁盘安全删除文件的能力。
- **极简 UI**：采用 Luminous 风格的浅色高级卡片式 UI，具备进度条交互和响应式支持。

## 技术栈
- **后端**：Node.js, Express, Axios (固定在 0.27.2 版本以适配 pkg 打包)。
- **前端**：Vanilla JavaScript (无框架), 原生 CSS3, HTML5。
- **系统工具**：集成 `child_process` 实现与操作系统的强交互（打开文件、展示文件夹等），使用 `pkg` 支持打包为各平台单体可执行文件 (.exe / 命令行工具)。

## 部署与运行架构
1. **本地开发/单机使用**：直接 `npm run dev` 启动，通过 `localhost:3000` 访问。
2. **服务器化/NAS 部署**：可打包为 Docker 镜像，部署在如飞牛 OS (fnOS) 这样的 x86 服务器上，作为个人私有化离线下载服务，手机/PC 均可通过局域网 Web 访问。
