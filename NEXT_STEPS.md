# 下一步计划 (Next Steps)

## 1. 登录态与同步体验优化 (高优先级)
- **登录持久化**：优化 Cookie 过期检测逻辑，支持自动检测并提醒用户重新扫码。
- **自定义同步深度**：允许用户设置从 50 条同步到 100/200 条。
- **NAS 版本同步适配**：探索在无头 Docker 环境下运行 Playwright 的最佳实践，实现 NAS 端的自动化收藏同步。

## 2. 网页 UI 移动端优化 (中优先级)
- 优化 PWA (Progressive Web App) 支持，添加 `manifest.json` 与 `Service Worker`，让手机访问时能完美呈现原生体验。
- 梳理手机端的输入栏交互。

## 3. 功能增强 (低优先级)
- **批量压缩导出**：小红书图集下载自动打包为 ZIP 供导出。
- **多任务队列控制**：引入简单的任务队列并发量控制器。
- **自定义 Cookie 注入**：为应对极端限制，增加手动填入 Cookie 的高级入口。
